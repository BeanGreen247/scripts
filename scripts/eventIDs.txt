Keyboard Event ID: event8
Mouse Event ID: event24
Make sure to adjust your kVM XML/settings for input devices

For mouse

<input type="evdev">
  <source dev="/dev/input/event24"/>
</input>

For Keyboard

<input type="evdev">
  <source dev="/dev/input/event8" grab="all" repeat="on"/>
</input>

just make sure to copy into the right places

incase the vm does not want to power down use the powerDownAllVMs.sh script that can be found in the scripts folder and can be executed like this

bash scripts/powerDownAllVMs.sh
